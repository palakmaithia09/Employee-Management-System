# Employee Management System (C++)

## Description
A file-handling based employee management system developed in C++. It allows adding, searching, and displaying employee records stored in binary files.

## Features
- Add employee details
- Display all employee records
- Search employee by ID and password
- Persistent data storage using files

## Technologies Used
- C++
- File Handling
- Structures

## How to Run
1. Compile the program using a C++ compiler.
2. Run the executable file.
3. Use menu options to manage employee data.

## Learning Outcome
- Understanding of file handling in C++
- Improved knowledge of data persistence
